	База данных broker может работать как с мобильным приложением, так и с сайтом
брокерской компании для организации покупки/продажи ценных бумаг (акции, облигации, металлы и так далее) на фондовых биржах (NASDAQ, NYSE, Московская  биржа, СПБ биржа). База данных создана в рамках работы над курсовым проектом по предмету "Основы реляционных баз данных. MySQL" и не отражает полной картины покупки/продажи ценных бумаг, а лишь затрагивает часть функционала по осуществлению брокерской деятельности.
